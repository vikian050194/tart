## About

**tart** is a tool to create and manage tags for images

## Motivation

**tart** is a pet-project that helps me to gain experience in Java.

Also there are few tools for image viewing and tag assignment, but I need something free, simple and a bit different features.

## Requirements

Developed and tested on Ubuntu 20.04.4 LTS and OpenJDK 17.0.6.
